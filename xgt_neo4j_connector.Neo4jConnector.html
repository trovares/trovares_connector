copy_data_from_neo4j_to_xgt(xgt_schemas, use_bolt=True) → None¶

Copies data from Neo4j to the requested vertex and/or edge frames in Trovares xGT.

This function copies data from Neo4j to xGT for all of the nodes and all of the relationships, one frame at a time.

Parameters

  • xgt_schemas (dict) – Dictionary containing schema information for vertex and edge frames to create in xGT. This dictionary can be the value returned from the get_xgt_schema_for() method.

  • use_bolt (boolean) – Use bolt when transferring data from Neo4j to python instead of arrow. By default this is true.

Return type

None

create_xgt_schemas(xgt_schemas, append=False) → None¶

Creates vertex and/or edge frames in Trovares xGT.

This function first infers the schemas for all of the needed frames in xGT to store the requested data. Then those frames are created in xGT. Finally, all of the nodes and all of the relationships are copied, one frame at a time, from Neo4j to xGT.

Parameters

  • xgt_schemas (dict) – Dictionary containing schema information for vertex and edge frames to create in xGT. This dictionary can be the value returned from the get_xgt_schema_for() method.

  • append (boolean) – Set to true when the xGT frames are already created and holding data that should be appended to. Set to false when the xGT frames are to be newly created (removing any existing frames with the same names prior to creation).

Return type

None
